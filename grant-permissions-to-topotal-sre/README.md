# Topotal SREに対する権限付与

Cloud Shellチュートリアルに沿ってコマンドを実行することで、Topotal SREに対して権限を付与することができます。

ブラウザ側で `net::ERR_BLOCKED_BY_CLIENT` エラーが生じて Cloud Shell が起動しない場合は、広告をブロックするブラウザ拡張機能を無効化した上でアクセスしてください。
