# Topotal SREに対する権限付与

Cloud Shellチュートリアルに沿ってコマンドを実行することで、Topotal SREに対して権限を付与することができます。

Cloud Shell が起動しない場合は、ブラウザのコンソール上で `net::ERR_BLOCKED_BY_CLIENT` エラーが生じているかどうかを確認してください。このエラー起因の問題は、広告をブロックするブラウザ拡張機能を無効化することで解決できます。
